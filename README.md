A simple Python web app for teachers to manage student grades and for students to view them.

## Features

* **Teacher Admin:** Full access to add, update, or delete student grades.
* **Student View:** Read-only access to view grades safely.
* **Secure Login:** Keeps accounts safe using encrypted passwords.
* **Database Safety:** Protects data using secure SQL queries.

## Tech Stack

* Python 3
* Flask
* SQLite3
